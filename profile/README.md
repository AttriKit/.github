# AttriKit

第一方广告归因系统，帮助广告主完全掌控归因数据。

## 关于 AttriKit

AttriKit 是面向国内广告主的第一方广告归因系统。我们帮助广告主在自己的服务器上部署归因系统，摆脱对广告平台和第三方归因服务的依赖。

### 核心价值

- **数据完全掌控** - 所有数据存储在您自己的服务器
- **成本优势明显** - 一次性部署，成本降低 80%+
- **开箱即用** - 完整的管理后台，快速部署
- **合规无忧** - 数据不出服务器，符合合规要求
